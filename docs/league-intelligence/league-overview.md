# League Overview

## Purpose
The League Overview page provides a real-time, competition-wide snapshot of league standings, team performance, and contextual trends. It is the entry point for exploring league intelligence.

## League Table Structure
- **Columns**: Position, Team, Played, Wins, Draws, Losses, Goals For, Goals Against, Goal Difference, Points, Form (last 5), Streaks.
- **Sorting**: Default by Points, then Goal Difference, then Goals For.
- **Filters**: By season, competition, and round (if applicable).

## Context Panels
- **League Trends**: Short summaries of league-wide stats (e.g., average goals per match, longest unbeaten run).
- **Upcoming Fixtures**: Key upcoming matches.
- **Recent Results**: Most recent match outcomes.

## Navigation
- Clicking a team row navigates to that team's profile page.
- Context panels may link to filtered views or deeper analytics.

## Non-Goals
- Does NOT show player-level stats or analytics.
- Does NOT provide event-level breakdowns (e.g., shots, xG).
- Does NOT allow editing of match data from this page.
- Is NOT a club-owned dashboard.
